# <多媒体计算机技术> 专选课作业

## 开发环境

- 系统环境: `Arch Linux`
- 浏览器版本: `Google Chrome 79` 

## 运行方式

- Huffman编码及解码

```
用浏览器打开 `huffman.html` (推荐使用chromium内核的浏览器确保兼容性)
```

- PCM脉冲编码调制
```
用浏览器打开 `pcm.html` (推荐使用chromium内核的浏览器确保兼容性)
```
